------------------------------------------------------------

Adicione todos os 9 arquivos "pdf" da Lista Python para a pasta de nome "Listas Python Brasil".

Com o repositório no VSCode abra o terminal integrado e envie essas alterações para o repositório git online através dos comandos abaixo:
    git add .
    git commit -m "Listas Python Brasil"
    git push origin main

Acesse seu repositório online e verifique como ficaram as alterações.

------------------------------------------------------------

Volte para o VSCode com o repositório aberto e crie um arquivo vazio, de nome ".gitkeep" dentro das outras 9 pastas.

Com o repositório no VSCode abra o terminal integrado e envie essas alterações para o repositório git online através dos comandos abaixo:
    git add .
    git commit -m "Estrutura de pastas inicial"
    git push origin main

Acesse seu repositório online e verifique como ficaram as alterações.

------------------------------------------------------------

Volte para o VSCode e inicie a resolução das listas.

Crie um arquivo de nome "01.js" dentro da pasta "1-EstruturaSequencial", e neste arquivo escreva o código de resolução para o primeiro exercício da primeira lista.

Para testar seu código, abra o terminal do VSCode, entre na pasta da respectiva lista e use o node para executar o arquivo:
    cd 1-EstruturaSequencial
    node 01.js

------------------------------------------------------------

Após finalizar suas codificações, envie seus arquivos para o repositório git online.

Mas siga essas regras:
    Quando for enviar os códigos, defina a mensagem de commit da seguinte forma:
        Se os códigos forem da primeira lista: "Resolução 1-EstruturaSequencial"
        Se os códigos forem da segunda lista: "Resolução 2-EstruturaDeDecisao"
        E assim por diante.
